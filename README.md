# ccxtt

```bash
git clone https://github.com/tankakatan/ccxtt.git
git clone https://github.com/tankakatan/ccxt.git && cd ccxt && npm link && cd ../ccxtt
brew install watchman
npm install -g react-native-cli
npm install
npm link ccxt
npm start # optionally
react-native run ios # or run iOS project with xcode
```
